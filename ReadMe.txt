Procedure:
1) Visit the website from "https://www.qt.io/download#eval-form-modal" then complete the 
evaluation and download it version 13.0.0.
2) Login with email and password which is added in the evaluation.
3) Verify the email address.
4) In installation folder choose “Qt 6.7 for desktop development”.
5) Follow the menu at the end press on button “Finished” and open program.
6) Press “open project.”
7) Choose From project file "CMakeLists"
8) At the end this screen will be opened, and press run on the left side. 
9) The main screen of the program will be shown
